# consultaCep
React Native - Consulta CEP


1.Install Node.js

2.```git clone https://github.com/ronaldsilva-ti/consultaCep.git```

3.```cd consultaCep```

4.```npm install```

5.```npx react-native start ```

5.```npx react-native run-android ```

6.For IOS ```npx react-native run-ios```
